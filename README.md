# Volunteer Library

### 准备工作
* 在MySql中创建名为account的数据库，然后再执行account.sql导入数据库<br>
* 用MyEclipse打开项目<br>
* 再项目中导入mysql的连接jar包<br>
* 运行项目即可<br>
